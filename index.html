API Documentation

Integrate AI text detection into your applications with our REST API. Here's how to get started:

Endpoint

HTTP 
POST https://your-api-url.onrender.com/predict

Python Example

Python 
import requests

url = "https://your-api-url.onrender.com/predict"
data = {
    "text": "Your text to analyze here...",
    "min_words": 50
}

response = requests.post(url, json=data)
result = response.json()

print(f"Prediction: {result['prediction']}")
print(f"Word Count: {result['word_count']}")

JavaScript Example

JavaScript 
fetch('https://your-api-url.onrender.com/predict', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        text: 'Your text to analyze here...',
        min_words: 50
    })
})
.then(response => response.json())
.then(data => {
    console.log('Prediction:', data.prediction);
    console.log('Word Count:', data.word_count);
});
